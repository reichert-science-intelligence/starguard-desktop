- **When**: Interventions due within timeframe

- **When**: Significant change in closure rate
- **Example**: "Closure Rate Dropped 15.2%"
- **Action**: Investigate root cause and adjust strategy

## âš™ï¸ Configuration Tips

### Star Rating Threshold
- **85%**: Standard threshold for most measures
- **90%**: Stricter threshold for high-priority measures
- **80%**: More lenient for challenging measures

### Opportunity Value Threshold
- **$10,000**: Good starting point
- **$25,000**: Focus on larger opportunities
- **$5,000**: Catch smaller opportunities

### Deadline Days Ahead
- **30 days**: Standard planning window
- **14 days**: More urgent focus
- **60 days**: Longer planning horizon

### Anomaly Threshold
- **15%**: Standard sensitivity
- **10%**: More sensitive to changes
- **20%**: Less sensitive, fewer false positives
