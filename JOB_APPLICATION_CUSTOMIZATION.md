# Job Application Customization Guide

## Cursor Prompt Template for Cover Letters

### Basic Template

```
Generate a cover letter for [Company] [Position] emphasizing:

1. My secure healthcare AI expertise (HIPAA-compliant LLM deployment)

2. Specific project: [HEDIS Optimizer OR Glycemic Model OR Secure Chatbot]

3. How I solve their pain point: [extracted from job description]

4. Quantified impact: Choose from $148M savings, 93% recall, 2.8-4.1x ROI

5. Close with: My approach bridges the gap between AI innovation and 
   healthcare compliance that [Company] needs for [specific initiative]

Keep to 3 paragraphs, 250 words max. Professional but conversational tone.
```

## Pain Point Extraction Guide

### Common Healthcare AI Pain Points

1. **Compliance Concerns**
   - "Maintaining HIPAA compliance"
   - "Protecting patient data"
   - "Regulatory requirements"
   - "Compliance barriers"

2. **AI Adoption Challenges**
   - "Implementing AI solutions"
   - "Leveraging LLM capabilities"
   - "AI innovation"
   - "Adopting new technologies"

3. **Data Security**
   - "Data security"
   - "Patient privacy"
   - "Protected health information"
   - "Data protection"

4. **Technical Challenges**
   - "Building predictive models"
   - "Data analytics"
   - "Machine learning deployment"
   - "AI infrastructure"

5. **Business Impact**
   - "Driving business value"
   - "ROI optimization"
   - "Cost savings"
   - "Efficiency gains"

### How to Extract Pain Points

1. **Read Job Description Carefully**
   - Look for problem statements
   - Identify challenges mentioned
   - Note compliance requirements
   - Highlight technical needs

2. **Identify Keywords**
   - HIPAA, compliance, security
   - AI, ML, analytics, data
   - Healthcare, patient, PHI
   - Innovation, transformation

3. **Match to Your Solutions**
   - Secure AI deployment → Compliance concerns
   - Local LLM → AI adoption challenges
   - On-premises processing → Data security
   - Quantified impact → Business value

## Complete Customization Workflow

### Step 1: Analyze Job Description

1. **Extract Pain Points**
   - Read job description carefully
   - Identify challenges mentioned
   - Note compliance requirements
   - Highlight technical needs

2. **Identify Keywords**
   - HIPAA, compliance, security
   - AI, ML, analytics, data
   - Healthcare, patient, PHI
   - Innovation, transformation

3. **Match to Your Solutions**
   - Map pain points to your projects
   - Select relevant metrics
   - Choose appropriate project

### Step 2: Generate Cover Letter

1. **Use Cursor Prompt**
   - Fill in company name
   - Fill in position title
   - Specify project to highlight
   - Include extracted pain point
   - Select quantified impact

2. **Customize Content**
   - Adjust tone to company culture
   - Reference specific initiatives
   - Connect to job requirements
   - Personalize closing

### Step 3: Review & Refine

1. **Check Length**
   - Target: 200-250 words
   - 3 paragraphs maximum
   - Concise and focused

2. **Verify Content**
   - Security/compliance mentioned first
   - Quantified impact included
   - Pain point addressed
   - Clear value proposition

3. **Proofread**
   - Grammar and spelling
   - Professional tone
   - No jargon without context
   - Clear and readable
