# QR code image

Place your portfolio landing QR code image here:

- **Filename:** `QR_-Landing.png`
- **Usage:** Referenced in the app footer as `src="QR_-Landing.png"` (Shiny serves files from `www/` at the root URL).

If you use a different filename (e.g. `qr-landing.png`), update the footer HTML in `app.py` to match.
