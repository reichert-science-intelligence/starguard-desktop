# Cursor Prompt: Cover Letter Generator

## Ready-to-Use Prompt Template

Copy and paste this prompt into Cursor, filling in the bracketed sections:

```
Generate a cover letter for [Company] [Position] emphasizing:

1. My secure healthcare AI expertise (HIPAA-compliant LLM deployment)

2. Specific project: [HEDIS Optimizer OR Glycemic Model OR Secure Chatbot]  

3. How I solve their pain point: [extracted from job description]

4. Quantified impact: Choose from $148M savings, 93% recall, 2.8-4.1x ROI

5. Close with: My approach bridges the gap between AI innovation and 
   healthcare compliance that [Company] needs for [specific initiative]

Keep to 3 paragraphs, 250 words max. Professional but conversational tone.
```

---

## Example Usage

### Input:
```
Generate a cover letter for Acme Healthcare Data Scientist emphasizing:

1. My secure healthcare AI expertise (HIPAA-compliant LLM deployment)

2. Specific project: HEDIS Optimizer

3. How I solve their pain point: Need to implement AI solutions for Medicare Advantage analytics while maintaining HIPAA compliance

4. Quantified impact: 2.8-4.1x ROI

5. Close with: My approach bridges the gap between AI innovation and 
   healthcare compliance that Acme Healthcare needs for Medicare Advantage optimization

Keep to 3 paragraphs, 250 words max. Professional but conversational tone.
```

### Expected Output:
A cover letter that:
- Leads with security/compliance
- Mentions specific project (HEDIS Optimizer)
- Addresses their pain point (Medicare Advantage + HIPAA)
- Includes quantified impact (2.8-4.1x ROI)
- Closes with value proposition

## Quick Fill-In Guide

### [Company]
- Extract from job posting
- Use full company name
- Example: "Acme Healthcare" or "UnitedHealth Group"

### [Position]
- Exact job title from posting
- Example: "Healthcare Data Scientist" or "ML Engineer"

### [HEDIS Optimizer OR Glycemic Model OR Secure Chatbot]
- **HEDIS Optimizer**: Use for analytics/optimization roles
- **Glycemic Model**: Use for ML/predictive modeling roles
- **Secure Chatbot**: Use for NLP/LLM/RAG roles

### [extracted from job description]
- Read job description carefully
- Identify main challenge mentioned
- Example: "Implementing AI while maintaining HIPAA compliance"
- Example: "Building predictive models for healthcare outcomes"

### [Choose from $148M savings, 93% recall, 2.8-4.1x ROI]
- **$148M savings**: For large-scale impact stories
- **93% recall**: For ML/performance-focused roles
- **2.8-4.1x ROI**: For analytics/business value roles

### [specific initiative]
- Extract from job description
- Company's stated goals/projects
- Example: "Medicare Advantage optimization"
- Example: "Care outcome prediction"
- Example: "Risk stratification"

## Project Selection Guide

### Use HEDIS Optimizer When:
- Role involves analytics/optimization
- Company works with Medicare Advantage
- Position requires ROI/business impact focus
- Job mentions HEDIS or quality measures

**Key Metrics:**
- 2.8-4.1x ROI
- $66K quarterly net benefit
- 12 HEDIS measures optimized

---

### Use Glycemic Model When:
- Role involves predictive modeling
- Company focuses on chronic disease management
- Position requires ML/statistical expertise
- Job mentions diabetes or screening

**Key Metrics:**
- 93% recall
- 87% precision
- $380K annual value

---

### Use Secure Chatbot When:
- Role involves NLP/LLM deployment
- Company wants AI-powered interfaces
- Position requires RAG/vector search expertise
- Job mentions natural language processing

**Key Metrics:**
- Zero external API calls
- On-premises vector search
- HIPAA-compliant RAG
