## Mobile-Specific Optimizations

### Performance

- **Caching**: Data cached for 5 minutes

## Troubleshooting

### Page Not Loading

- Check internet connection
- Clear browser cache
- Try refreshing
- Use WiFi if available

### Data Not Updating

- Click "Refresh Data" button
- Wait 5 minutes for cache to expire
- Check last updated timestamp

### Touch Not Working

- Ensure you're tapping buttons (not hovering)
- Use full-width buttons
- Check browser zoom level (should be 100%)

### Text Too Small

- Mobile CSS should auto-apply
- Check browser zoom settings
- Ensure mobile view is enabled
