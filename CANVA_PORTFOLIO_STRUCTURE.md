# Canva Portfolio Site Structure & Copy

## Design Guidelines

**Colors:** Healthcare-appropriate (blues #0066CC, greens #00A86B, whites #FFFFFF)
**Fonts:** Modern sans-serif (Montserrat, Open Sans) for headers, readable serif for body
**Layout:** Single-page scrollable, mobile-responsive
**Visual Style:** Clean, data-driven, professional

## MOBILE RESPONSIVENESS CONSIDERATIONS

- Stack sections vertically on mobile
- Reduce font sizes by 20-30%
- Make buttons full-width or larger touch targets
- Simplify diagrams for smaller screens
- Ensure icons and images scale appropriately

---

## PRINT/PDF EXPORT TIPS

- Use high-contrast colors (avoid light grays)
- Ensure all text is readable at print size
- Include page breaks at logical sections
- Add page numbers if multi-page
- Test print preview before sharing
