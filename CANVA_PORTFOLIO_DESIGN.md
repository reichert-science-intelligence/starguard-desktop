## Canva Template Instructions

### Step 1: Create Security Architecture Visual

1. **Create New Design**: 8.5" x 11" (Letter)
2. **Add Title**: "Bridging the Healthcare AI Gap"
3. **Create Two Columns**: Left (Traditional), Right (Secure)
4. **Add Icons**: Red X for left, Green checkmarks for right
5. **Add Arrows**: Show data flow (left: outward, right: circular)
6. **Add Bottom Message**: Centered, bold text
7. **Apply Color Scheme**: Red/Orange (left), Green/Blue (right)

### Step 2: Create Project Impact Dashboard

1. **Create New Design**: 11" x 17" (Tabloid)
2. **Add Title**: "Healthcare AI Projects"
3. **Create Three Columns**: One for each project
4. **Add Project Cards**: Rounded rectangles with borders
5. **Add Icons**: Security, Impact, Technology for each
6. **Apply Color Scheme**: Blue, Green, Purple for projects
7. **Add Metrics**: ROI, recall, value prominently displayed

### Step 3: Export & Use

1. **Export as PDF**: High resolution (300 DPI)
2. **Export as PNG**: For web use (1920px width)
3. **Save to Portfolio Folder**: Organize by project
4. **Add to LinkedIn**: Featured section
5. **Include in Presentations**: Portfolio pitch deck
